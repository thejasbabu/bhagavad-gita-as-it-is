# Bhagavad Gita

## Steps to run locally

1. Install the gitbook cli
    ```
      npm install -g gitbook-cli
    ```
1. Install dependencies
    ```
     gitbook install
    ```
1. Build the book
    ```
    gitbook build .
    ```
1. Run it locally
    ```
     gitbook serve . --port 4001
    ```

Vist localhost:4001/ in the browser of your choice.
